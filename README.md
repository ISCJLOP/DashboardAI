# LA TRAMPA DE LA FALSA SEGURIDAD: EFICACIA, SESGOS Y DILEMAS ÉTICOS DE LAS HERRAMIENTAS DE DETECCIÓN DE IA EN LA INVESTIGACIÓN.

**Estudio de caso: El ecosistema de detección y parafraseo de QuillBot.**

Este repositorio contiene el código fuente del dashboard analítico e interactivo desarrollado como parte integral de la tesis, diseñado para visualizar, segmentar y transparentar los resultados empíricos obtenidos en el estudio sobre detectores generativos y sus sesgos.

### Descripción del Dashboard
El dashboard funge como una herramienta estadística que expone las falencias metodológicas de los detectores automatizados evaluados, destacando particularmente:
* El colapso predictivo en textos 100% generados por Inteligencia Artificial (falsos negativos consistentes).
* La visualización del sesgo de base del sistema, que aparenta eficacia total analizando únicamente controles pre-2022.
* La perspectiva ética desde los Principios de Heredia y la Declaración de Chapultepec sobre el uso probatorio de estas herramientas.

### Aspectos Técnicos
El proyecto ha sido desarrollado como una Single Page Application (SPA) haciendo uso de:
* **React** + **TypeScript** para la lógica del sistema
* **TailwindCSS** para un diseño moderno (glassmorphism/neumorphism oscuro) 
* **Recharts** para mapeo e interpretación gráfica interactiva
